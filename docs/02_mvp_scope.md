# MVP Scope

## Версия

`v0.1 Route Viewer`

## Что входит в MVP

### Импорт GPX

- импорт `.gpx`;
- поддержка tracks/routes/waypoints;
- обработка нескольких трек-сегментов;
- имя маршрута из файла или metadata;
- базовая валидация.

### Карта

- отображение маршрута поверх карты;
- старт и финиш;
- авто-зум по bbox маршрута;
- переключение темы карты в будущем;
- базовый слой через MapLibre.

### Статистика

- дистанция;
- набор высоты;
- сброс высоты;
- минимальная/максимальная высота;
- количество точек;
- примерная длительность в будущем профиле.

### Высотный профиль

- график высоты по дистанции;
- подсветка выбранного места на карте;
- сегменты по километрам;
- поиск резких градиентов.

### Сегменты

- разбивка по 1 км или по естественным участкам;
- дистанция сегмента;
- средний градиент;
- набор/сброс;
- предупреждения.

### Экспорт

- повторный экспорт GPX без разрушения базовой структуры;
- позже: экспорт GeoJSON/CSV отчёта.

## Что не входит в MVP

Чтобы не распухнуть до недоделанного Komoot, в v0.1 не делаем:

- собственный роутинг;
- собственный tile renderer;
- аккаунты;
- подписки;
- социальную ленту;
- turn-by-turn навигацию;
- офлайн весь мир;
- сложное редактирование маршрута мышкой;
- полноценный OSM импорт;
- мультипользовательский backend.
