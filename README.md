# Komoot Killer MVP

Локальный и быстрый GPX viewer / route inspector для вело и гревела.

Идея MVP: сначала не строим собственный роутер. Сначала делаем удобную читалку GPX с картой, статистикой, высотным профилем и базовой диагностикой маршрута. После этого добавляем OSM-анализ покрытия, потом редактор, и только затем экспериментируем с роутингом.

## Главный принцип

Не писать свой рендер карт и не пытаться сразу сделать свой Valhalla. Используем готовые решения для карт и рендера, а собственную ценность создаём в Rust-ядре: анализ GPX, сегменты, метрики, профиль высоты, предупреждения, скоринг и будущий routing layer.
